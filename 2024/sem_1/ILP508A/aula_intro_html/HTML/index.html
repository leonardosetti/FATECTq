Uma página HTML (Hyper Text Markup Language)

Um breve texto:

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

"Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

Usando Tabelas

Estrutura de tabelas em HTML
Nome da TAG Função da TAG Exemplo
TAG: <TABLE>

A tag table define uma tabela para o HTML

Podendo ser atribuída a uma classe ou id específicos

<table>

[Conteúdo da Tabela]

</table>
TAG: <TR>

TR significa Table Row (linha de tabela)

Permite que seja inserida uma nova linha para a tabela

Deve ser usada dentro de uma TAG TABLE

<table>

<tr>

conteúdo da linha #1 da tabela

</tr>

conteúdo da linha #2 da tabela

</tr>

conteúdo da linha #3 da tabela

</tr>

conteúdo da linha #N da tabela

</tr>

</table>
TAG: <TH>

TH significa Table Header (Títulos de Colunas)

Permite que seja inserido um Título de Coluna para a tabela

Deve ser usada dentro de uma TAG TR

<table>

<tr>

<th>título da coluna #1</th>

<th>título da coluna #2</th>

<th>título da coluna #3</th>

<th>título da coluna #N</th>

</tr>

</table>
TAG: <TD>

TD significa Table Data (Dados da tabela)

Cada "TD" é equivalente a uma célula da tabela

Pode ou não conter uma informação, tais como:

Textos, Listas, outras Tabelas, Imagens, Links, etc

Deve ser usada dentro de uma TAG TR

<table>
