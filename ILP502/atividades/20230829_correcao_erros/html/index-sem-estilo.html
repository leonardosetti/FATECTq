Atividade de Correção de Erros

O propósito desta atividade é analisar e corrigir o maior número de erros.

  • Considerar erros no HTML
  • Considerar erros de sintaxe
  • Considerar erros de semântica

✔Relação dos elementos HTML a serem corrigidos:

  • A declaração do tipo de documento <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> está sendo usada, mas é mais apropriado usar o formato <!DOCTYPE html> para representar o HTML5.
  • A tag de fechamento </head> está ausente antes da abertura da tag <body>.
  • A tag </main> deve ser fechada antes das tags </footer> e </html>.
  • A tag de fechamento </body> está faltando.
  • A tag de abertura <main> está faltando sua tag de fechamento: </main>.
  • O elemento <p> está aninhado dentro do elemento <ul>, o que não é uma estrutura HTML válida.
  • A tag <script> no final do código está usando o atributo type = "txt/javascript", que deve ser corrigido para type = "text/javascript".
  • A tag <span> não está sendo usada corretamente, pois não está dentro de um contexto adequado.

✔Relação dos erros de sintaxe:

  • Dentro da tag <script>, os métodos Alert e confirme devem ser escritos em letras minúsculas como alert e confirm.
  • A tag <script> no final do código tem uma chamada de função incompleta, faltando um parêntese de fechamento.
  • Tags de Fechamento Ausentes: Várias tags <td> estão sem suas respectivas tags de fechamento </td>. As tags <td> para "height" não estão corretamente fechadas e devem ser separadas.
  • O símbolo &rec; não existe e deve ser corrigido para &reg que corresponde a ®.

✔Relação dos erros semânticos:

  • Apesar de não ser um erro, é bom incluir target="_blank" nas tags <a href=""></a>.
  • A tag <meta charset="ISO-8859-1" /> deve ser corrigida para <meta charset="UTF-8"> para definir o conjunto de caracteres corretamente.
  • Os elementos <footer> e <header> devem ser usados para conteúdo de rodapé e cabeçalho, respectivamente.
  • A tag <footer> não está dentro da estrutura correta, devendo estar dentro da tag <body>.
  • A tag <main> está sendo usada duas vezes consecutivas, mas seu uso não está adequado, já que ela deve ser usada para representar o conteúdo principal de uma página.

Alguns Atributos utilizados em elementos da Linguagem HTML


Atributo Descrição
class Atribui uma classe ao elemento
id Atribui um identificador ao elemento - um id deve ser único
style Permite incluir estilos
lang Define o idioma principal do elemento
alt Define um texto alternativo
width Define uma largura para o elemento
height Define uma altura para o elemento
