Projeto: Livraria Digital DigLib
Objetivo

O projeto de livraria digital, denominado Dig Lib, tem como objetivo disponibilizar obras literárias, livros, artigos, conhecimento acadêmico e outros materiais digitais para acesso público.

A plataforma permitirá que usuários cadastrados tenham acesso a esses materiais de forma flexível, a qualquer hora e em qualquer lugar.

Benefícios:
  • Acesso a informações de forma rápida e fácil
  • Redução de custos com mídia física
  • Preservação de materiais raros e históricos
  • Buscas precisas
  • Disponibilidade global
Requisitos Funcionais:
  • Inclusão de dados
  • Cadastro de usuários
  • Pagamentos/compras
  • Manutenção de acervo
  • Relatórios
  • Certificação Digital
Requisitos Não Funcionais:
  • Linguagem Java
  • Multiplataforma
  • Facilidade de uso
  • Disponibilidade de Informações cadastrais offline
ABORDAGEM: Manutenção do Sistema DigLib

Com base nesses critérios, as boas práticas de manutenção para o sistema Dig Lib devem incluir as seguintes ações:

Manutenção preventiva

A manutenção preventiva deve ser realizada regularmente para evitar problemas e garantir a disponibilidade e o desempenho do sistema.

  • Atualização de softwares e hardwares
  • Correção de bugs
  • Melhoria de desempenho
  • Implementação de novas funcionalidades
Manutenção Corretiva

A manutenção corretiva deve ser realizada para corrigir problemas que ocorrem no sistema.

As atividades de manutenção corretiva incluem:

  • Diagnóstico de problemas
  • Implementação de correções
  • Teste de correções
Manutenção Evolutiva

A manutenção evolutiva deve ser realizada para melhorar o sistema e atender às novas necessidades dos usuários.

As atividades de manutenção preventiva incluem:

  • Adição de novas funcionalidades
  • Melhoria da usabilidade
  • Adaptação a novos requisitos
